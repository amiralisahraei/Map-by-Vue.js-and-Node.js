<h1>Make a simple by Vue.js and Node.js</h1>

In this project we send an ajax request from Vue to Node to take the data.

<h4>How to setup dev environment</h4>
1- cd server <br>
2- npm install <br>
3- node server <br>
4- cd .. <br>
5- npm install <br>
6- npm run sever <br>
7- open browser on port 8080 <br>
